# Real--Time-Sign-Language-Detection-CNN
Real time sign language detection using CNN and Computer Vision

This project implements a real-time sign language detection system
using Convolutional Neural Networks (CNN) and computer vision.

## Features
- Real-time hand gesture detection
- CNN-based sign classification
- Webcam-based input
- Live prediction output

## Technologies Used
- Python
- OpenCV
- TensorFlow / Keras
- CNN

## Applications
- Assisting hearing and speech impaired individuals
- Human-computer interaction
- Accessibility tools

## Project Status
- In progress / Learning phase
